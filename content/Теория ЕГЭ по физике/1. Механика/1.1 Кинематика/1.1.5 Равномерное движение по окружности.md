---
title: 1.1.5 Криволинейное движение. Движение по окружности
---
**==Криволинейное движение==** — движение, при котором траектория материальной точки является кривой линией.

При криволинейном движении вектор скорости в каждой точке направлен по **касательной** к траектории. Если траектория — окружность радиуса $R$, то движение удобно описывать углом поворота радиуса.

![Схема движения точки по окружности](<Теория ЕГЭ по физике/assets/1.1.5-dvizhenie-po-okruzhnosti.svg>)

## Угловая и линейная скорость
**==Линейная скорость==** $v$ показывает, как быстро точка проходит путь вдоль окружности.\
**==Угловая скорость==** $\omega$ показывает, на какой угол поворачивается радиус за единицу времени.

Для движения по окружности линейная и угловая скорости связаны формулой:
$$
\boxed{v = \omega R}
$$
где $R$ — радиус окружности.

При **равномерном** движении по окружности угловая скорость постоянна:
$$
\boxed{\omega = \frac{2\pi}{T} = 2\pi \nu}
$$
где $T$ — период обращения, то есть время одного полного оборота;\
$\nu$ — частота обращения, то есть число оборотов за единицу времени.

## Центростремительное ускорение
При движении по окружности направление скорости всё время меняется. Поэтому даже при постоянном модуле скорости ускорение не равно нулю.

**==Центростремительное ускорение==** направлено по радиусу к центру окружности:
$$
\boxed{a_{\text{цс}} = \frac{v^2}{R} = \omega^2 R}
$$

`Важно!` При равномерном движении по окружности скорость постоянна только по модулю. Вектор скорости не постоянен, потому что его направление меняется.

## Полное ускорение
В общем случае криволинейного движения полное ускорение удобно раскладывать на две составляющие:
$$
\vec{a} = \vec{a}_{\tau} + \vec{a}_n
$$
$\vec{a}_{\tau}$ — тангенциальное ускорение, оно направлено по касательной и отвечает за изменение модуля скорости;\
$\vec{a}_n$ — нормальное ускорение, оно направлено к центру кривизны и отвечает за изменение направления скорости.

Для равномерного движения по окружности $a_{\tau} = 0$, поэтому полное ускорение совпадает с центростремительным:
$$
\vec{a} = \vec{a}_{\text{цс}}
$$

---
Коротко:
1) Скорость при движении по окружности направлена по касательной.
2) Связь линейной и угловой скоростей: $v = \omega R$.
3) При равномерном движении по окружности $\omega = \frac{2\pi}{T} = 2\pi \nu$.
4) Центростремительное ускорение направлено к центру: $a_{\text{цс}} = \frac{v^2}{R} = \omega^2 R$.
5) Полное ускорение складывается из тангенциальной и нормальной составляющих.
